# INSTALL.md
## Prerequisites
To install and run the cs2-game-tool, you will need to have a Windows operating system and Python installed on your machine. The tool utilizes the Windows API and DirectX for overlay functionality, so ensure that your system meets the necessary requirements. Additionally, familiarity with Python scripting is recommended for advanced customization and development.

## Download
To obtain the latest version of the cs2-game-tool, visit the releases page on the GitHub repository: https://github.com/Dimensioneisense/cs2-game-tool/releases. Download the appropriate package for your system, taking note of the specific requirements and dependencies listed for each release.

## Install
The installation process for the cs2-game-tool involves the following steps:
1. Extract the downloaded package to a directory of your choice.
2. Run the installation script, following the prompts to complete the setup.
3. Ensure that all dependencies, including Python and DirectX, are installed and up-to-date.

## First run
After installation, launch the cs2-game-tool to begin exploring its features. The tool provides a user-friendly interface for creating custom game modes, as well as a range of options for modifying and extending the game's functionality. Refer to the documentation and tutorials provided within the repository for guidance on getting started with the tool.

## Troubleshooting
If you encounter issues during installation or while running the cs2-game-tool, consider the following steps:
1. Verify that your system meets the prerequisites and dependencies listed for the tool.
2. Check the GitHub repository's issue tracker for known problems and solutions.
3. Consult the documentation and community resources for guidance on resolving specific errors or issues.
By following these steps and referring to the available resources, you should be able to successfully install and utilize the cs2-game-tool to enhance your cs2 gaming experience.
