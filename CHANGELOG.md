# Changelog
All notable changes to the cs2-game-tool project will be documented in this file.

## [v1.0.0] - 2024-07-26
### Added
* Support for creating custom game modes using a Python scripting interface
* Integration with the Windows API to enable advanced game automation features
* DirectX-based overlay system for displaying game information and statistics
* Game module loader for easy installation and management of community-created modules
* Utility functions for automating common game tasks and workflows
* Windows desktop integration for seamless interaction with the game tool
* Support for loading and executing custom Python scripts within the game environment

### Notes
* The initial release of the cs2-game-tool is focused on providing a solid foundation for game development and automation on the Windows platform
* The tool is designed to be highly extensible, with a modular architecture that allows developers to easily create and share custom game modules and scripts
* The cs2-game-tool is compatible with the latest version of the cs2 game and is available for download on GitHub at https://github.com/Dimensioneisense/cs2-game-tool
