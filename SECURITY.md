# Security
The cs2-game-tool project takes security seriously and appreciates your help in keeping the project secure. 

## Supported versions
The following versions of cs2-game-tool are currently supported with security updates:
| Version | Supported          |
|---------|--------------------|
| latest  | : Yes              |
| previous| : No               |

## Reporting a vulnerability
If you believe you have found a security vulnerability in cs2-game-tool, please report it to us via the [Issues page](https://github.com/Dimensioneisense/cs2-game-tool/issues). When submitting a vulnerability report, please provide as much detail as possible, including steps to reproduce the issue and any relevant logs or screenshots.

## Disclosure policy
The cs2-game-tool project adheres to a policy of responsible disclosure. This means that we will work with you to verify and address the vulnerability as quickly as possible, and we will publicly disclose the vulnerability and its resolution once the fix has been released. We appreciate your cooperation and patience in this process, and we are committed to maintaining the security and integrity of the cs2-game-tool project.
